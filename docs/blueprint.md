# **App Name**: VotoTablet

## Core Features:

- Candidate Display: Display the list of candidates, with name, photo and party.
- Candidate Selection: Allow the voter to select a candidate from the list via a tap.
- Confirmation Screen: Display the selected candidate name and photo for confirmation before submitting the vote.
- Vote Submission: Send the vote via HTTP request to the ESP32.
- Voting help: Generate informative messages helping the voter decide, acting as a tool to increase confidence in the voter. 

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and stability, similar to traditional voting booths. The deep blue color is also inspired by governmental buildings around the world, creating a recognizable reference to the application.
- Background color: Light gray (#F0F0F0), a desaturated version of the primary color. It is used to create a neutral, clean and non-distracting interface.
- Accent color: Soft orange (#FFAB40), an analogous color, provides visual contrast and can be used for active buttons or selected states.
- Font: 'Inter', a sans-serif font, for clear readability across different screen sizes. It is used to improve the legibility in user interfaces.
- Use simple and clear icons to represent actions or status, to enhance the overall user experience.
- Use a clear, simple and accessible layout that supports both touch and screen reader input.
- Use subtle transitions to create an experience that is polished, elegant and reassuring.
